# essential-oil-cdn

Image CDN for the [essential-oil-guide](https://github.com/kiolllery8-dev/essential-oil-guide) website (https://intelliverse.tw).

All 499 images are served via **jsDelivr** global CDN:

```
https://cdn.jsdelivr.net/gh/kiolllery8-dev/essential-oil-cdn@main/images/{filename}
```

Example:
- `https://cdn.jsdelivr.net/gh/kiolllery8-dev/essential-oil-cdn@main/images/banner-lavender.jpg`
- `https://cdn.jsdelivr.net/gh/kiolllery8-dev/essential-oil-cdn@main/images/hero-home.png`

## Contents
- 499 botanical banner images (TopView AI-generated, 2K 21:9)
- Hero covers, brand logos, article thumbnails
